# DAO DApp Shared Investment

This repository contains a full-stack DAO DApp for shared investments, including:

- **Smart Contracts** (Solidity, Hardhat)
- **Backend API** (TypeScript, Node.js, Prisma)
- **Frontend** (React, Vite)
- **Infrastructure** (Postgres, Nginx, Docker)

## Prerequisites

- Node.js >= 22
- npm >= 10
- Docker & Docker Compose
- Hardhat
